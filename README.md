<h1 align=center>HiteTool</h1>
<center>鸿合科技开发的辅助工具</center>


这是从[鸿合科技](www.hitevision.com)的[软件](https://pie.hitecloud.cn/pie)中提取出来的内容。这里面有倒计时、高光等内容。
[![](https://i.loli.net/2020/01/30/bPquLhgvKkNUBef.jpg)](https://sm.ms/image/bPquLhgvKkNUBef)
`HiteClock.exe`集成了时钟、倒计时和秒表。
[![](https://i.loli.net/2020/01/30/v65YNTEIC4ctiSR.jpg)](https://sm.ms/image/v65YNTEIC4ctiSR)
`HiteLens.exe`具有放大作用。
[![](https://i.loli.net/2020/01/30/VRwJhycaLKbnsQW.jpg)](https://sm.ms/image/VRwJhycaLKbnsQW)
`HiteSpotlight.exe`能聚焦某个地方。
